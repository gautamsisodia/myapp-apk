Workout Tracer – AI Fitness Assistant App
📌 Overview
Workout Tracer is an AI‑powered fitness tracking application designed to help users log workouts, monitor progress, and achieve their health goals.
Developed as part of the Kaggle VibeCoding Agents Capstone (Concierge Agents Track), this project showcases practical AI integration in a mobile/web app environment.

🚀 Features
📖 Workout Logging – Record exercises, sets, reps, and weights.
🔥 Streaks & Personal Bests – Stay motivated with streak tracking and milestone records.

📊 Progress Charts – Visualize growth with intuitive graphs.
🎯 Goal Tracking – Set fitness goals and monitor achievements.
🤖 AI Coaching (Gemini 3.5 Flash) – Personalized insights, smart recommendations, and recovery guidance.
🏅 Gamification – Badges, streaks, and community sharing for motivation.

🛠️ Tech Stack
Frontend: Responsive UI (HTML, CSS, JavaScript)
Backend: Secure data handling & APIs
Database: Structured schema for workouts, goals, and streaks
Deployment: Google Cloud Run
AI Integration: Gemini 3.5 Flash for personalized coaching
